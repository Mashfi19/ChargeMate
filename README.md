## рџ“– Overview

ChargeMate is a modern EV charging platform that combines premium UI/UX with intelligent charging management. The project demonstrates how a digital charging ecosystem can integrate charger hardware, charging stations, smart scheduling, digital payments, vehicle management, and charging analytics into one seamless web application.

The website is built using **HTML, CSS, and Vanilla JavaScript** with a strong focus on performance, responsiveness, accessibility, and interactive user experiences.
